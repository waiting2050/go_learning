## 注意事项

1. 所有接口文档均采用 OpenAPI 3.0.1 规范
2. ID 字段必须为 string 类型
3. 密码需要进行 bcrypt 加密
4. 热门排行榜接口必须使用 Redis 缓存
5. 支持分页的接口需要正确处理 page_num 和 page_size 参数
6. 必须实现双 Token 机制（Access-Token 和 Refresh-Token）
