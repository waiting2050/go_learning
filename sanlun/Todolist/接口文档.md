# 用户

## POST 注册新用户

POST /user/register

> Body 请求参数

```json
{
  "user_name": "Tester003",
  "password_digest": "123456"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|Authorization|header|string| 否 |none|
|body|body|object| 是 |none|

> 返回示例

```json
{
  "status": 400,
  "msg": "用户名已存在",
  "data": null
}
```

```json
{
  "status": 200,
  "msg": "注册成功",
  "data": null
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|
